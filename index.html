Welcome to GoDownload manager!

This is simple download manager support multipart download and resume. For add, start|stop, remove download use web-gui

A couple of comments:

  • To run service in cmd just run godownload
  • Setting save in ./godownload. You can edit setting in json forma
  • All files stored in /Downloads
